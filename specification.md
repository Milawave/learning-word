# Спецификация проекта "Учу слово"

## 1. Описание проекта
Учебный проект мобильного приложения для изучения иностранных слов. Приложение позволяет пользователю:

- Добавлять новые слова и наборы слов
- Учить слова по интервальной методике (алгоритм Лейтнера)
- Просматривать прогресс и статистику изучения
- Вести активный словарь и архив выученных слов

> Данный проект выполнен в рамках учебного курса аналитики.  
> Он демонстрирует навыки анализа требований, построения UML-диаграмм и проектирования API.  
> Реальная разработка приложения и интеграция с внешними сервисами не выполнена.

---

## 2. Пользователи

| Класс пользователя   | Возможности |
|---------------------|-------------|
| Неавторизованный    | Просмотр слова на выбранном языке с переводом |
| Авторизованный      | Доступ ко всем функциям: добавление слов, просмотр активного словаря, статистика прогресса |

---

## 3. Основные функции

**Функции с высоким приоритетом:**
1. Расширение словарного запаса (добавление слов/наборов слов)
2. Интервальные повторения (по алгоритму Лейтнера)
3. Отслеживание прогресса и статистика изучения

**Дополнительные функции:**
- Прослушивание аудио и транскрипции слов
- Добавление иллюстраций к словам
- Возможность переноса слов в архив

---

## 4. Основные пользовательские сценарии (Use Case)

### UC-1: Авторизация
- Пользователь авторизуется через соцсеть или номер телефона  
- После успешной авторизации получает доступ ко всем функциям

### UC-2: Добавление слова
- Пользователь выбирает язык и добавляет новое слово в активный словарь  
- Система предлагает перевод, транскрипцию, аудио и примеры  
- Слово сохраняется в активном словаре

### UC-3: Интервальное повторение
- Пользователь учит слова по схеме: часто → раз в неделю → раз в месяц → иногда  
- Если пользователь ошибается, слово возвращается в активный цикл повторения

### UC-4: Перенос слова в архив
- Пользователь отмечает слово как выученное  
- Система переносит слово в архив и сохраняет статистику

---

## 5. Алгоритмы и логика

**Алгоритм повторений:**
- Используется метод Лейтнера для интервального повторения слов  
- Активный словарь ограничен 100 словами  
- Статистика ведёт учёт правильных и неправильных ответов  

---

## 6. Нефункциональные требования

- **Интерфейс:** простой, интуитивно понятный, поддержка мультимедиа  
- **Производительность:** стабильная работа при подключении к интернету, время отклика сервера ≤ 5 секунд  
- **Безопасность:** хранение пользовательских данных, HTTPS  
- **Расширяемость:** проект предусматривает возможность расширения функционала  

---

## 7. REST API (учебное проектирование)

**Основные endpoints:**

| Метод | URL | Описание |
|-------|-----|----------|
| GET   | /Language | Получить текущий язык |
| PUT   | /Language/{code} | Изменить язык |
| GET   | /Languages | Список языков |
| GET   | /ActiveDictionary | Список изучаемых слов |
| POST  | /ActiveDictionary/word | Добавить слово |
| PATCH | /ActiveDictionary/word | Перенести слово в архив |
| DELETE | /Archive | Очистить архив |

> API оформлено в формате OpenAPI 3.0 как учебное проектирование.  
> Примеры JSON-ответов подготовлены для демонстрации структуры и логики приложения.  
> Массивы ошибок включены как учебные примеры обработки исключений.
